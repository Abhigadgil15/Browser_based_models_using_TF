# Running models on browser using tensorflow javascript
To create models based on tensorflow and implement them on any browser.

Requirements:-
Web Server using any web application development IDE. Recommended using Brackets.

- For the rock paper scissor spock and lizard prediction run the rpsls.html and share the access of webcam on browser.
- Get at least 50-100 live samples for rock, paper,scissors, spock and lizard then train the model.
- Once the model starts predicting accurately, download the model and you can use samples of another category to customise the model with different signs.
